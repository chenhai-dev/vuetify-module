# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-08

### Added
- Initial release
- Tree-shaking via vite-plugin-vuetify
- Auto-import components, directives, and labs
- Multiple icon set support (MDI, FA, SVG variants)
- Blueprint configuration (md1, md2, md3)
- Locale support with multiple languages (en, fr, es, de)
- SSR support with proper hydration
- Theme management composables
- Custom composables (useVTheme, useSnackbar, useConfirmDialog, useLoading)
- Date adapter support
- Prefetch and preload optimizations

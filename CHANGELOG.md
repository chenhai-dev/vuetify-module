## [1.0.0] - 2025-01-01

### Added

- Theme Management with cookie persistence for easy theme switching
- Component Defaults management for global and scoped component default props
- Performance optimizations with tree-shaking and bundle optimization
- Custom composables: `useVTheme` for theme management and `useVDefaults` for component defaults
- Presets support for quick component style presets (denseForm, comfortableForm, rounded, flat, elevated)
- Custom hooks for configuration modification via `nuxt-vuetify:config` hook
- Nuxt 4 Ready with support for `app/` directory structure
- SASS/SCSS customization support
- SSR configuration options
- Blueprint preset support (md1, md2, md3)
- Lab components support
